By using the pythagorean theorum, we can find that \(c=\sqrt{1 + f'(x_k^*)^2} \Delta x\text{.}\)

Remark 1.3.5.
\begin{align*} c^2 \amp = (\Delta x)^2 + (f'(x_k^*)\Delta x)^2\\ c \amp = \sqrt{ \Delta x^2 + (f'(x_k^*)\Delta x)^2}\\ \amp = \sqrt{(\Delta x)^2 (1 + (f'(x_k^*))^2}\\ \amp = \sqrt{\Delta x^2} \sqrt{1 + (f'(x_k^*)^2}\\ \amp = \Delta x \sqrt{1 + (f'(x_k^*)^2}\\ \amp = \sqrt{1 + (f'(x_k^*)^2} \Delta x \end{align*}
